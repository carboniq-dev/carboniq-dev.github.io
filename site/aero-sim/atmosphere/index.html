    Atmosphere Actor¶

    This section offers a summary of the properties of the Atmosphere Actor that impact the calculations related to atmospheric composition.

    Atmosphere¶

    Property Type Description
    Is Active Boolean Toggle to activate the atmosphere simulation.

    Calculation Options¶

    Property Type Description
    Calc Humidity Boolean Toggle to calculate humidity effects.
    Calc Pressure Boolean Toggle to calculate atmospheric pressure effects.
    Calc Temperature Boolean Toggle to calculate temperature effects.
    Use Advanced Influence Boolean Use advanced algorithms for influence calculations.
    Use Boiling Point Boolean Checks if the temperature at the center of the Gas Vent or Absorber is above the boiling point.
    Report On Location Changed Boolean Report changes when location is changed.

    Debug¶

    Property Type Description
    Show Debug Panel Boolean Displays a debug panel showing the atmosphere measurement values.

    Composition¶

    Property Type Description
    Base Gas Concentrations Map Base concentrations for different gas types.
    Gas Concentration Caps Map Caps on concentrations for different gases.

    Example¶

    Influence¶

    Property Type Description
    Influence Unbound Boolean If true, the influence is always active, regardless of distance.
    Influence Radius Float This parameter defines the spatial extent of the atmospheric region, specifying the maximum distance from the central point or source within which atmospheric effects are applied.
    Apply Atmospheric Boundary Falloff Boolean Activates simulation of decreasing impact with increasing distance from the atmosphere's center.
    Atmospheric Boundary Falloff Factor Float Determines the rate at which atmospheric effects decrease towards the outer boundary. Higher values result in a sharper decrease.
    Show Bounds Boolean Determines whether to show the boundary area visually.
    Bounds Color LinearColor Color of the indicator for the boundary area.

    Height¶

    Property Type Description
    Start Height Float Starting altitude for atmospheric calculations.
    Max Height Float Maximum altitude for atmospheric effects.
    Height Effect On Gas Factor Float Determines how much height affects the gas concentration.

    Temperature¶

    Property Type Description
    Temperature Float Planets average temperature.
    Temperature Cap Float The maximum cap for temperature in the atmosphere calculation.
    Temp Effect On Gas Factor Float Determines how much temperature affects the gas concentration.

    Humidity¶

    Property Type Description
    Humidity Float Planets average humidity.
    Humidity Effect On Gas Factor Float Determines how much humidity affects the gas concentration.

    Gravity¶

    Property Type Description
    Gravity Float Planets average gravity.
    Gravity Effect On Gas Factor Float Determines how much gravity affects the gas concentration.

    Pressure¶

    Property Type Description
    Pressure Cap Float The maximum cap for atmospheric pressure in the atmosphere calculation.
    Pressure Multiplier Float A modifier that adjusts the rate of atmospheric pressure buildup, used to either increase or decrease the resulting pressure.
